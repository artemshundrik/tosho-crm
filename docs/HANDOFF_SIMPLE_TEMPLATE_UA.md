## 4. Мінімальний план, якщо щось сталося

### Якщо треба терміново відновити систему

1. Зайти в `1Password`
2. Відкрити записи:
   - `Tosho CRM / Supabase`
   - `Tosho CRM / Netlify`
   - `Tosho CRM / Dropbox`
   - `Tosho CRM / Backups`
   - `Tosho CRM / Recovery Notes`
3. Перевірити:
   - чи працює Supabase
   - чи працює Netlify
   - чи доступні резервні копії
4. Якщо проблема в деплої:
   - дивитися Netlify
5. Якщо проблема в базі або файлах:
   - дивитися Supabase і Backups
6. Якщо потрібне повне відновлення:
   - брати інструкцію з restore
   - використовувати backup credentials

## 8. Що не можна робити

- не зберігати ключі в чатах
- не передавати паролі через Telegram або email
- не покладатися тільки на `.env` файли на одному ноутбуці
- не тримати єдину копію доступів у голові однієї людини

## 9. Мій рекомендований мінімум

Якщо зовсім спростити, то тобі обов'язково треба заповнити хоча б це:

- `Supabase`: логін, пароль, admin URL, service role key
- `Netlify`: логін, пароль, admin URL
- `Dropbox`: логін, пароль, backup folder, refresh token якщо використовується
- `Backups`: де лежать бекапи і чим їх відновлювати
- `Recovery Notes`: 5-10 рядків людською мовою що де знаходиться

Цього вже достатньо для адекватної передачі.
